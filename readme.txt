A beautiful and interactive goal progress tracker that displays progress as a horizontal thermometer with customizable gradient colors.

== Description ==

The Goal Progress block allows you to create stunning visual representations of your goals and achievements. Perfect for fundraising campaigns, project milestones, fitness goals, or any metric you want to track and display on your website.

Key Features:

* **Horizontal Thermometer Design** - Beautiful visual representation of progress
* **Customizable Goal Label** - Add a descriptive label for your goal
* **Flexible Value Input** - Set minimum, maximum, and current values
* **Dual Display Modes** - Show progress as actual numbers or percentage
* **Gradient Colors** - Choose from beautiful gradient presets or create custom gradients
* **Real-time Preview** - See changes instantly in the editor
* **Responsive Design** - Looks great on all screen sizes

Whether you're tracking donations, sales targets, project completion, or personal goals, this block makes it easy and beautiful.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/goal-progress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the Goal Progress block to any post or page using the block editor
4. Customize your goal settings in the block inspector panel

== Frequently Asked Questions ==

= Can I use custom colors for the thermometer? =

Yes! The block comes with several beautiful gradient presets, and you can also create your own custom gradient using the color picker in the inspector panel.

= Can I display the progress as a percentage? =

Absolutely! You can toggle between showing actual numbers (e.g., "$5,000 of $10,000") or percentage (e.g., "50%") using the display format option.

= Is the block responsive? =

Yes, the Goal Progress block is fully responsive and will adapt beautifully to any screen size.

= Can I have multiple goal trackers on the same page? =

Yes! You can add as many Goal Progress blocks as you need, each with its own settings and values.
